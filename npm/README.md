# cllmhub

Turn your local LLM into a production API.

## Install

```bash
npm install -g cllmhub
```

Or run directly:

```bash
npx cllmhub --help
```

## Usage

```bash
cllmhub login
cllmhub publish
cllmhub whoami
```

For more information, visit [github.com/cllmhub/cllmhub-cli](https://github.com/cllmhub/cllmhub-cli).
